# ОТКЛИК24 — Автоотклик на HH.ru (Chrome Extension)

**ОТКЛИК24** — это браузерное расширение, которое автоматизирует процесс отклика на вакансии на HeadHunter (hh.ru). Оно помогает экономить время, отправляя отклики с подготовленными шаблонами писем, соблюдая лимиты и имитируя действия человека.

## 🚀 Возможности
- **Автоматические отклики**: Работает в фоновом режиме на странице поиска вакансий.
- **Шаблоны писем**: Сохраняйте до 3 вариантов сопроводительных писем и переключайтесь между ними в один клик.
- **Умные лимиты**: Устанавливайте дневной лимит откликов (с авто-сбросом каждый день), чтобы избежать блокировок.
- **Умный пропуск тестов**: Вакансии с тестовым заданием или доп. вопросами пропускаются (детектор смотрит только в окно отклика, а не по всей странице).
- **Авто-релокация**: Автоматически подтверждает отклик на вакансии в другой стране.
- **Чёрный список**: Пропуск нежелательных компаний/слов.
- **Анти-бот задержки**: Случайные паузы между откликами имитируют действия человека.
- **История**: Экспорт истории откликов в CSV.
- **Современный UI**: Тёмная тема, удобный интерфейс.

## 📦 Установка (Режим разработчика)

Т.к. расширение еще не опубликовано в магазине, его можно установить вручную:

1. Скачайте этот проект (Code -> Download ZIP) и распакуйте, или склонируйте репозиторий.
2. Откройте в браузере Chrome страницу управления расширениями: `chrome://extensions/`.
3. В правом верхнем углу включите **"Режим разработчика"** (Developer mode).
4. Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked).
5. Выберите папку проекта: `otlik24`.

## 🛠 Использование

1. Перейдите на сайт [hh.ru](https://hh.ru) и настройте фильтры поиска вакансий.
2. Откройте расширение (нажмите на иконку в панели браузера).
3. Введите или выберите текст сопроводительного письма.
4. Установите дневной лимит (рекомендуется не более 100-200).
5. Нажмите **"Запустить автоотклик"**.
6. Расширение начнет работу. Не закрывайте вкладку с поиском (можно переключиться на другую, но вкладка должна быть открыта).

## 📝 Разработка и Публикация

Для упаковки расширения (`.crx`):
1. Убедитесь, что в папке `icons/` лежат файлы: `icon-16.png`, `icon-32.png`, `icon-48.png`, `icon-128.png`.
   - Если их нет, используйте `tools/generate-icons.html` для генерации.
2. В `chrome://extensions/` нажмите **"Упаковать расширение"** (Pack extension).
3. Укажите путь к корневой папке проекта.
4. (Опционально) При обновлении укажите существующий `.pem` ключ.

## 🔒 Приватность
Расширение работает локально в вашем браузере. Данные шаблонов и история хранятся в `localStorage` вашего браузера и никуда не отправляются.
