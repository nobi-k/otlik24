# Инструкция по сборке ОТКЛИК24

Это расширение не требует компиляции (нет webpack, babel и т.д.). Оно состоит из исходных файлов HTML, CSS и JavaScript. Процесс "сборки" представляет собой просто упаковку этих файлов в ZIP-архив для распространения.

## Настройка окружения
- **ОС**: Windows 10/11 (или любая ОС с поддержкой Python)
- **Инструменты**: Python 3.12+ (используется для скрипта упаковки)

## Исходные файлы
Расширение состоит из следующих файлов:
- `manifest.json`
- `content.js`
- `popup.html`
- `popup.js`
- `icons/` (папка с иконками PNG/SVG)

## Шаги сборки
Чтобы создать дистрибутив (`otlik24_store.zip`):

1. **Откройте терминал** в корневой папке проекта.

2. **Запустите скрипт упаковки**:
   ```bash
   python pack_extension.py
   ```

   *Примечание: Этот скрипт (`pack_extension.py`) включен в исходный код. Он просто создает ZIP-архив с необходимыми файлами и гарантирует, что пути к файлам используют прямые слеши (`/`) для совместимости со всеми платформами и браузерами.*

3. **Результат**:
   - Скрипт создаст файл `otlik24_store.zip` в той же директории.
   - Этот ZIP-файл является готовым к загрузке в магазин расширений.
